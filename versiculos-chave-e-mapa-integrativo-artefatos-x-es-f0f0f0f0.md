VERSÍCULOS-CHAVE E MAPA INTEGRATIVO
Artefatos Cognitivos x Sabedoria Bíblica

DRIVER PRINCIPAL: O Olhar Fixo e o Coração Íntegro
Definição: O direcionamento deliberado, sustentado e unificado da atenção —
olhos, mente, coração e vontade — ao que verdadeiramente importa, resistindo
à dispersão que fragmenta não apenas a produtividade mas a própria alma.

VERSÍCULO 1: Provérbios 4:25-27 (Manifesto Operacional)
"Os teus olhos olhem para a frente, e as tuas pálpebras olhem diretamente
diante de ti. Pondera a vereda de teus pés, e todos os teus caminhos sejam
retos. Não declines nem para a direita nem para a esquerda."

Mapeamento frase a frase:
- "Olhos olhem para a frente" (nabat) -> Deep Work (foco deliberado)
- "Pálpebras olhem diretamente" (yashar) -> Environment Design (filtrar)
- "Pondera a vereda dos pés" (palles) -> Time Blocking (examinar rotinas)
- "Não declines para direita ou esquerda" -> Attention Residue (eliminar desvios)
- "Retira o teu pé do mal" -> Pomodoro (resistir ativamente aos estímulos)

VERSÍCULO 2: Mateus 6:22-23 (Tese Central)
"Se os teus olhos forem bons [haplous], todo o teu corpo terá luz."
Haplous = singular, inteiro, sem fratura. O olho focado ilumina tudo.
Poneros = torcido, desviado. O olho fragmentado escurece tudo.
A qualidade da atenção determina a qualidade de TUDO.

VERSÍCULO 3: Hebreus 12:1-2 (Protocolo de Foco)
"Deixemos todo embaraço [onkos]... olhando para Jesus [aphorao]."
1. Eliminar peso (onkos) -> Environment Design
2. Identificar pecado circundante (euperistatos) -> Attention Residue
3. Correr com paciência (hypomone) -> Pomodoro (resistência treinada)
4. Fixar o ponto focal (aphorao) -> Deep Work (absorção total)
5. Manter na temporalidade (carreira proposta) -> Time Blocking

VERSÍCULO 4: Tiago 1:6-8 (Diagnóstico)
"O homem de coração dobre [dipsychos] é inconstante [akatastatos]
em TODOS os seus caminhos."
Dipsychos = duas almas operando simultaneamente.
Akatastatos = não estabelecido, não assentado. Turbulência perpétua.
A instabilidade não é pontual — é sistêmica. Afeta TODOS os caminhos.

VERSÍCULO 5: Filipenses 3:13-14 (Protocolo Paulino)
"Uma coisa faço [hen de]: esquecendo-me [epilanthanomai] das coisas
que atrás ficam, avançando [epekteinomai] para as que estão diante."
1. Hen de ("uma coisa") -> Deep Work (foco radical)
2. Epilanthanomai ("esquecendo") -> Attention Residue (descarregar resíduo)
3. Epekteinomai ("estirando-se") -> Flow / absorção total
4. Skopos ("alvo") -> Time Blocking (estruturar o percurso)

VERSÍCULO 6: Salmo 86:11 (Oração do Foco)
"Unifica meu coração [yachad levavi] para temer o Teu nome."
Cal Newport pede: "Concentre-se em uma tarefa por vez."
O salmista pede: "Unifica meu coração." A gramática é diferente —
o pedido é o mesmo.

VERSÍCULO 7: Lucas 10:41-42 (Contraste Supremo)
"Marta, Marta, estás ansiosa [merimna] e afadigada com muitas coisas,
mas uma só é necessária. Maria escolheu a boa parte."
Marta = merimna (ansiedade que fragmenta) = Shallow Work frenético
Maria = atenção unificada na UMA coisa = Deep Work espiritual
