SISTEMA INTEGRADO DOS 5 ARTEFATOS DE CONCENTRAÇÃO
Visão Unificada — A Lente que Gera Fogo

A mesma luz solar dispersa que aquece sem transformar pode, concentrada através
de uma lente num único ponto, gerar fogo. Cada artefato é um ajuste na lente.

PROTOCOLO EM 5 PASSOS:

[ ] PASSO 1: PROJETAR O AMBIENTE (Environment Design)
    "O espaço está projetado para foco?"
    - Físico: local dedicado, mesa limpa, celular longe
    - Digital: notificações off, apps bloqueados
    - Social: horários comunicados, sinais visuais

[ ] PASSO 2: DEFINIR O QUE FOCAR (Deep Work)
    "Esta tarefa cria valor que é difícil de replicar?"
    - SIM -> Deep Work. Proteja o bloco
    - NÃO -> Shallow Work. Agrupe num bloco menor

[ ] PASSO 3: AGENDAR QUANDO FOCAR (Time Blocking)
    "Quando este bloco de Deep Work acontece?"
    - Bloco agendado no calendário. Inegociável
    - Deep Work nos melhores horários (manhã)

[ ] PASSO 4: MANTER O FOCO (Pomodoro Technique)
    "Como mantenho a concentração durante o bloco?"
    - Timer de 25 min (ou 50, ou 90 — conforme nível)
    - Interrupções internas: anote e volte
    - Pausa obrigatória a cada ciclo

[ ] PASSO 5: PROTEGER ENTRE TAREFAS (Attention Residue)
    "Como minimizo o resíduo quando preciso trocar?"
    - Completar antes de trocar (preferível)
    - Ritual de transição (anotar, fechar, pausar)
    - Brain dump se necessário

DIAGNÓSTICO RÁPIDO:
- Se você não sabe NO QUÊ focar -> comece pelo Deep Work
- Se você sabe no quê mas nunca ACHA TEMPO -> comece pelo Time Blocking
- Se você agenda mas NÃO AGUENTA o bloco -> comece pelo Pomodoro
- Se você troca demais de TAREFA -> comece pelo Attention Residue
- Se seu AMBIENTE sabota -> comece pelo Environment Design
