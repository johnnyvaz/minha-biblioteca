ENVIRONMENT DESIGN FOR FOCUS — O ARQUITETO DO ESPAÇO
BJ Fogg (Stanford Behavior Design Lab) + James Clear (Atomic Habits)
Fontes: Tiny Habits (2020), Atomic Habits (2018), Digital Minimalism (2019)

REVELAÇÃO CENTRAL:
Força de vontade é recurso FINITO, depletável e não confiável. Cada decisão
consciente (resistir tentação, forçar concentração) consome do mesmo reservatório.
Quando esgota — geralmente à tarde — a resistência colapsa.

PRINCÍPIO:
Projete o ambiente para que o foco seja o caminho de MENOR resistência.

FRICTION DESIGN:
  ADICIONAR FRICÇÃO              REMOVER FRICÇÃO
  para distração:                para foco:
  +------------------+          +------------------+
  | Celular em       |          | Material de      |
  | outro cômodo     |          | trabalho pronto  |
  |                  |          |                  |
  | Deslogar de      |          | Rituais de       |
  | redes sociais    |          | início automát.  |
  |                  |          |                  |
  | Remover apps     |          | Ambiente         |
  | do celular       |          | silencioso       |
  |                  |          |                  |
  | Notificações     |          | 1 tarefa visível |
  | OFF (padrão)     |          | por vez          |
  |                  |          |                  |
  | Fechar TUDO que  |          | Lugar específico |
  | não for a tarefa |          | = foco associado |
  +------------------+          +------------------+

3 CAMADAS DE AMBIENTE:
1. Físico: local, mesa, luz, som, temperatura, celular longe
2. Digital: notificações off, apps bloqueados, abas fechadas, modo foco
3. Social: sinais de "não interrompa", horários comunicados, cultura de respeito

EQUAÇÃO COMPORTAMENTAL (BJ Fogg):
Comportamento = Motivação x Habilidade x Gatilho
Se qualquer uma é zero, o comportamento não acontece.
Para o foco: manipule o AMBIENTE (gatilhos + fricção) em vez de depender de motivação.

5 PERGUNTAS-CHAVE:
1. "Quantos passos separam minha mão do celular agora?" — se <2, distração vence
2. "Meu ambiente foi projetado para foco ou para distração?"
3. "As pessoas ao meu redor sabem que estou em modo de foco?"
4. "Quantas notificações recebo por hora — e quantas são urgentes?"
5. "Se um desconhecido visse meu espaço, diria que é lugar de foco?"

ARMADILHAS:
- Design sem execução: planejar o ambiente perfeito e nunca implementar
- Investir em equipamento em vez de processo: mesa ergonômica + celular ao lado
- Controlar ambiente mas não mente: espaço perfeito + mente indisciplinada
- Ignorar o ambiente social: as pessoas não são telepatas — comunique

APLICAÇÕES:
- Trabalho: open office? Headphones + horários comunicados + sala reservada
- Casa: lugar específico exclusivo para trabalho profundo (nem que seja uma cadeira)
- Fé: "cantinho de oração" — o cérebro associa aquele espaço com presença de Deus
- Hábitos: celular longe + notificações off = a mudança mais barata e mais eficaz

EXEMPLO HISTÓRICO: Marcel Proust revestiu as paredes do quarto com cortiça para
bloquear som. Escrevia deitado, cortinas fechadas, à noite. Resultado: 3.000
páginas de Em Busca do Tempo Perdido — Environment Design radical.

EXEMPLO BÍBLICO: Jesus e o lugar deserto (Mc 1:35) — "Saiu e foi para um lugar
deserto, e ali orava." Três elementos: (1) saiu do ambiente de demandas,
(2) lugar sem estímulos, (3) função única. O Filho de Deus não confiava na
força de vontade — confiava no design do ambiente.
